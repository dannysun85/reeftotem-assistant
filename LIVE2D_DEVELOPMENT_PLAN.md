# AI数字人助手开发计划

## 项目概述

基于Tauri + React + Live2D + AI技术栈开发的智能数字人助手应用，包含**主控制窗口**和**Live2D宠物窗口**双窗口架构，实现语音交互、大模型对话、声音克隆和完整的桌面宠物体验。

**当前状态**: ✅ v0.1.2已完成 - Live2D桌面宠物功能完整实现，支持透明窗口、拖拽交互、眼神跟随、右键菜单系统

---

## 🎯 双窗口架构设计

### 技术栈整合
- **AI技术**: 大模型对话、语音识别、语音合成、声音克隆
- **Live2D渲染**: Live2D Cubism SDK + WebGL
- **桌面框架**: Tauri 2.x (多窗口支持)
- **前端框架**: React + TypeScript + Vite
- **语音处理**: Web Audio API + TTS/STT服务

### 窗口架构
```
主控制窗口 (main-window)
├── CleanSimpleChat (聊天界面)
│   ├── 对话历史记录
│   ├── 语音输入控制
│   ├── 实时语音波形显示
│   └── 设置面板
├── PersonaDebugger (模型管理)
│   ├── Live2D模型选择器
│   ├── 模型配置管理
│   └── 实时状态监控
└── AI配置面板
    ├── 大模型配置 (OpenAI, Claude, 本地模型)
    ├── 语音引擎设置
    ├── 声音克隆训练
    └── Agent行为配置

↓

Live2D宠物窗口 (live2d-window)
└── Live2DWindow组件
    └── RealLive2DComponent
        ├── Live2DCore加载
        ├── LAppDelegate (应用委托)
        ├── LAppLive2DManager (模型管理)
        ├── 唇形同步引擎
        └── Canvas渲染 (当前模型)
```

## 📋 技术架构和要求

### 核心技术栈
- **桌面框架**: Tauri 2.x (多窗口支持)
- **前端框架**: React 18 + TypeScript + Vite
- **Live2D渲染**: Cubism SDK + WebGL
- **AI技术**: OpenAI/Claude API + 本地模型支持
- **语音技术**: Web Speech API + 第三方TTS/STT服务
- **声音克隆**: ElevenLabs/Azure Speech集成

### 开发环境要求
- Node.js 18+ 和 pnpm 包管理器
- Rust 开发环境和 Tauri CLI
- 支持 WebGL 的现代浏览器
- AI服务API密钥 (OpenAI/Claude等)
- 语音服务账户 (Azure/Google等)

### 项目文件结构
```
src/
├── components/
│   ├── Live2D/           # Live2D组件层
│   ├── Chat/             # 聊天界面组件
│   ├── Settings/         # 设置配置组件
│   └── Voice/            # 语音处理组件
├── lib/
│   ├── live2d/          # Live2D核心框架
│   ├── ai/              # AI对话引擎
│   ├── voice/           # 语音处理模块
│   └── utils/           # 工具函数
├── services/            # 业务逻辑层
├── hooks/              # React Hooks
└── types/              # TypeScript类型定义

public/
├── assets/live2d/      # Live2D模型资源
├── models/             # 配置文件
└── live2d-window.html  # Live2D窗口入口

src-tauri/
├── src/                # Rust后端代码
├── capabilities/       # 权限配置
└── tauri.conf.json     # Tauri配置
```
