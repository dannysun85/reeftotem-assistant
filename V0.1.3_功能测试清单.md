## 📋 核心功能测试

### 1. Live2D模型显示
- [ ] 模型正常加载显示
- [ ] 模型动画正常播放
- [ ] 模型名称正确显示（应为配置文件中的名称，而非硬编码"cat"）

### 2. 鼠标交互功能
- [ ] **眼神跟随**：鼠标移动到模型区域时触发眼神跟随
- [ ] **指针样式**：鼠标悬停在模型上时显示手型指针
- [ ] **模型边界**：能清楚识别模型的交互边界
- [ ] **双击拖拽**：双击模型启动窗口拖拽功能
- [ ] **拖拽状态**：拖拽时鼠标指针变为抓取状态

### 3. 右键菜单功能
- [ ] **菜单显示**：右键点击模型显示设置菜单
- [ ] **菜单透明度**：右键菜单不受窗口透明度影响
- [ ] **切换模型**：点击切换模型选项（待实现）
- [ ] **眼神跟随开关**：可以开启/关闭眼神跟随
- [ ] **透明度滑块**：滑块可见且可调节窗口透明度（30%-100%）
- [ ] **窗口置顶**：可以切换窗口置顶状态
- [ ] **重置位置**：重置窗口到默认位置
- [ ] **关于对话框**：显示应用信息
- [ ] **退出应用**：正常退出应用

### 4. 性能优化功能
- [ ] **智能帧率控制**：60FPS活跃状态，30FPS空闲状态
- [ ] **调试信息清理**：移除不必要的console.log输出
- [ ] **错误日志保留**：保留重要的错误信息记录

### 5. 窗口管理功能
- [ ] **透明度设置**：窗口透明度可调节
- [ ] **置顶功能**：窗口可以保持置顶
- [ ] **位置重置**：可以重置窗口位置
- [ ] **跨平台支持**：在不同操作系统上正常运行
