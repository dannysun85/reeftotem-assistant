# 腾讯云语音服务业务流程快速开始指南

## 🚀 快速开始

### 步骤 1: 腾讯云服务配置

#### 1.1 开通语音服务
1. **登录腾讯云控制台**: [https://console.cloud.tencent.com/](https://console.cloud.tencent.com/)
2. **开通语音识别服务**:
   - 搜索"语音识别"
   - 点击"立即开通"
   - 选择"一句话识别"
   - 完成"实名认证"（如需要）
3. **开通语音合成服务**:
   - 搜索"语音合成"
   - 点击"立即开通"
   - 选择"基础语音合成"
4. **创建 API 密钥**:
   - 进入"访问管理" > "API密钥管理"
   - 点击"新建密钥"
   - 选择"编程访问"
   - 记录生成的 `SecretId` 和 `SecretKey`

#### 1.2 获取应用 ID
- **语音识别**: 在 ASR 控制台的"应用管理"中找到应用 ID
- **语音合成**: 在 TTS 控制台的"应用管理"中找到应用 ID（通常与 ASR 相同）

### 步骤 2: 本地配置

#### 2.1 配置环境变量
```bash
# 复制配置模板
cp .env.example .env

# 编辑配置文件
vim .env
```

#### 2.2 填写真实配置
```env
# 替换为您的真实配置
VITE_TENCENT_SECRET_ID=AKID您的真实SecretId
VITE_TENCENT_SECRET_KEY=您的真实SecretKey
VITE_TENCENT_REGION=ap-beijing  # 或选择其他区域
VITE_TENCENT_APP_ID=您的应用ID
```

#### 2.3 重启应用
```bash
# 关闭当前应用 (Ctrl+C)
pnpm tauri dev
```

### 步骤 3: 业务流程测试

#### 3.1 测试语音识别
1. **启动应用**: 应用启动后会在右下角显示 Live2D 角色
2. **打开语音界面**: 右键点击角色 → 选择"语音交互演示"
3. **开始录音**: 点击"🎤 开始录音"按钮
4. **说话测试**: 对着麦克风说话，建议：
   - 清晰发音
   - 语速适中
   - 录音时间 5-30 秒
5. **查看结果**: 停止录音后，系统会自动调用腾讯云 ASR API 并显示识别结果

#### 3.2 测试语音合成
1. **AI 对话**: 在文字聊天中输入消息
2. **自动合成**: AI 回复后，系统会自动调用腾讯云 TTS API
3. **语音播放**: 合成完成后自动播放语音
4. **参数调节**: 在语音界面中可以调节：
   - 音量 (0.0-1.0)
   - 语速 (0.2-2.0)
   - 音调 (-20.0-20.0)
   - 语音类型 (100+ 种选择)

## 🔧 业务流程详解

### 完整语音交互流程

```
用户说话 → 🎤 录音开始 → 实时波形显示 → 🎯 停止录音
     ↓
音频数据 → 腾讯云 ASR API → 📝 语音识别结果 → AI 处理 → 💬 文本回复
     ↓
文本内容 → 腾讯云 TTS API → 🔊 语音合成 → 🎧 音频播放
```

### 错误处理流程

```
API 调用失败 → 📋 错误日志 → 💡 用户友好提示 → 🔧 解决方案建议
     ↓
网络错误 → 检查网络连接
认证错误 → 验证密钥配置
限流错误 → 检查账户余额
参数错误 → 调整音频/文本参数
```

## 📊 成本控制

### 费用标准
- **语音识别**: ~¥0.6/次 (一句话识别)
- **语音合成**: ~¥2-3/万字符 (基础语音合成)

### 优化建议
1. **录音时长**: 控制在 30 秒内，提高识别准确率
2. **文本长度**: 单次合成 500 字符以内，降低成本
3. **缓存策略**: 重复文本使用缓存结果
4. **频率限制**: 避免过于频繁的 API 调用

### 监控方法
1. **查看日志**: 右键角色 → "开启调试模式" → 浏览器控制台
2. **用量统计**: 登录腾讯云控制台查看详细用量
3. **账单监控**: 定期检查账单，控制成本

## 🐛 常见问题解决

### Q1: API 调用失败，提示 "AuthFailure"
**解决方案**:
1. 检查 `SecretId` 和 `SecretKey` 是否正确
2. 确认 API 密钥有足够权限（支持 ASR 和 TTS）
3. 检查 `AppId` 是否对应正确服务

### Q2: 语音识别结果不准确
**解决方案**:
1. 确保录音环境安静，减少背景噪音
2. 使用正确的音频格式（16kHz，单声道）
3. 说话清晰，语速适中
4. 录音时长控制在 30 秒内

### Q3: 语音合成速度慢
**解决方案**:
1. 减少文本长度，分段处理长文本
2. 选择合适的语音类型
3. 使用音频缓存机制
4. 优化网络连接

### Q4: 应用启动后配置不生效
**解决方案**:
1. 检查 `.env` 文件是否在项目根目录
2. 确认配置格式正确（没有多余空格或特殊字符）
3. 重启应用使配置生效
4. 检查浏览器控制台是否有配置加载错误
