# 腾讯云语音服务配置指南

## 概述

本应用集成了腾讯云语音服务，包括语音识别(ASR)和语音合成(TTS)功能。本文档将指导您如何配置腾讯云服务以使用完整的语音交互功能。

## 功能特性

- **语音识别 (ASR)**: 将录制的语音转换为文本
- **语音合成 (TTS)**: 将文本转换为自然语音播放
- **实时交互**: 完整的语音对话体验
- **多种音色**: 支持100+种中文语音选择

## 前置要求

1. 腾讯云账号
2. 已开通语音识别和语音合成服务
3. 创建了对应的SecretId和SecretKey

## 配置步骤

### 1. 获取腾讯云API密钥

1. 登录[腾讯云控制台](https://console.cloud.tencent.com/)
2. 进入"访问管理 > API密钥管理"
3. 创建新的API密钥或使用现有密钥
4. 记录`SecretId`和`SecretKey`

### 2. 开通语音服务

#### 语音识别(ASR)
1. 在腾讯云控制台搜索"语音识别"
2. 开通"一句话识别"服务
3. 记录应用ID(`AppId`)

#### 语音合成(TTS)
1. 在腾讯云控制台搜索"语音合成"
2. 开通"语音合成"服务
3. 记录应用ID(`AppId`)(通常与ASR相同)

### 3. 配置环境变量

在项目根目录创建`.env`文件：

```env
# 腾讯云语音服务配置
VITE_TENCENT_SECRET_ID=your_secret_id_here
VITE_TENCENT_SECRET_KEY=your_secret_key_here
VITE_TENCENT_REGION=ap-beijing
VITE_TENCENT_APP_ID=your_app_id_here
```

### 4. 环境变量说明

| 变量名 | 必需 | 说明 | 示例值 |
|--------|------|------|--------|
| `VITE_TENCENT_SECRET_ID` | 是 | 腾讯云API密钥ID | AKIDxxxxxxxxxxxxxxxxxxxxxxxx |
| `VITE_TENCENT_SECRET_KEY` | 是 | 腾讯云API密钥Key | xxxxxxxxxxxxxxxxxxxxxxxx |
| `VITE_TENCENT_REGION` | 否 | 服务区域，默认ap-beijing | ap-beijing, ap-shanghai |
| `VITE_TENCENT_APP_ID` | 是 | 腾讯云应用ID | 1234567890 |

### 5. 支持的服务区域

- `ap-beijing` - 北京
- `ap-shanghai` - 上海
- `ap-guangzhou` - 广州
- `ap-chengdu` - 成都
- `ap-singapore` - 新加坡
- `ap-hongkong` - 香港

## 使用说明

### 语音识别配置

默认ASR配置：
- 采样率: 16000Hz
- 声道数: 1 (单声道)
- 引擎类型: 16k_zh (16kHz中文)
- 编码格式: WebM Opus

### 语音合成配置

默认TTS配置：
- 语音类型: 1001 (智逍遥 - 成熟女声)
- 音量: 1.0 (100%)
- 语速: 1.0 (正常速度)
- 音调: 0.0 (标准音调)
- 采样率: 16000Hz

### 可用语音类型

应用支持100+种中文语音，包括：

| ID | 名称 | 描述 |

| 1002 | 智瑜 | 成熟男声，适合播报 |
| 1003 | 智美 | 温柔女声，适合故事 |
| 1005 | 智莉 | 甜美女声，适合助理 |
| 1007 | 智芸 | 知性女声，适合教学 |

## 故障排除

### 1. 服务未配置错误

**错误**: `腾讯云语音服务未配置，请检查配置信息`

**解决**: 检查`.env`文件中的配置是否正确，确保所有必需变量都已设置。

### 2. API调用失败

**错误**: 语音识别或合成失败

**解决**:
- 检查腾讯云账号余额
- 确认已开通相应服务
- 验证API密钥权限
- 检查网络连接

### 3. 麦克风权限

**错误**: `无法访问麦克风`

**解决**:
- 确保浏览器有麦克风权限
- 检查系统麦克风设置
- 尝试重新授权麦克风权限

### 4. 音频播放失败

**错误**: `音频播放失败`

**解决**:
- 检查系统音量设置
- 确认浏览器音频权限
- 尝试不同浏览器

## API调用限制

腾讯云语音服务有调用限制：

- **一句话识别**: 通常按次数计费
- **语音合成**: 按字符数计费
- 具体限制请参考腾讯云官方文档

## 最佳实践

1. **安全**: 不要在前端代码中硬编码API密钥
2. **成本**: 监控API调用次数和费用
3. **性能**: 合理设置录音时长，避免过长音频
4. **用户体验**: 提供清晰的权限请求说明

## 技术实现

### 前端架构

```
VoiceInteractionDemo.tsx (主要UI)
├── TencentCloudVoiceService.ts (服务层)
├── AudioProcessor.tsx (音频处理)
└── VoiceRecorder.tsx (录音控制)
```

### 后端集成

```
src-tauri/src/lib.rs
├── tencent_asr (语音识别命令)
└── tencent_tts (语音合成命令)
```

### 数据流

```
录音 → MediaRecorder → ArrayBuffer → Tauri → 腾讯云API → 识别结果 → AI回复 → 语音合成 → 播放
```
