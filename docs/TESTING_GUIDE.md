# 语音交互功能测试指南

## 🎉 应用启动成功！

您的 Reeftotem Assistant 已经成功启动！现在可以测试语音交互功能了。

## 📋 测试准备

### 1. 配置腾讯云语音服务（可选）

如果您想使用真实的语音识别和合成功能：

1. **复制环境变量配置文件**：
   ```bash
   cp .env.example .env
   ```

2. **编辑 .env 文件**，填入您的腾讯云配置：
   ```env
   VITE_TENCENT_SECRET_ID=your_secret_id_here
   VITE_TENCENT_SECRET_KEY=your_secret_key_here
   VITE_TENCENT_REGION=ap-beijing
   VITE_TENCENT_APP_ID=your_app_id_here
   ```

3. **重启应用**：
   ```bash
   # 关闭当前应用 (Ctrl+C)
   # 然后重新启动
   pnpm tauri dev
   ```

### 2. 测试模式说明

- **Mock 模式**：无需配置即可测试基础功能和界面
- **真实模式**：配置腾讯云后可使用真实的语音识别和合成

## 🧪 功能测试步骤

### 步骤 1：基础界面测试

1. **找到 Live2D 角色**：应用启动后，Live2D 角色会显示在屏幕右侧
2. **右键菜单**：右键点击角色，查看上下文菜单
3. **调试模式**：在菜单中启用"开启调试模式"查看详细日志

### 步骤 2：语音录制测试

1. **右键点击 Live2D 角色**
2. **选择"语音交互演示"**
3. **在弹出的界面中**：
   - 点击"🎤 开始录音"按钮
   - 对着麦克风说话（建议：测试简单的问候语）
   - 观察实时波形可视化效果
   - 点击"⏹️ 停止录音"

### 步骤 3：语音识别测试

- **Mock 模式**：会显示模拟的识别结果
- **真实模式**：会调用腾讯云 API 进行真实识别

### 步骤 4：AI 对话测试

1. 语音识别完成后，系统会自动发送给 AI
2. 查看 AI 的回复内容
3. 检查对话历史记录

### 步骤 5：语音合成测试

1. AI 回复后，系统会自动进行语音合成
2. **Mock 模式**：播放模拟的音频
3. **真实模式**：播放腾讯云合成的真实语音

## 🔧 高级功能测试

### 音频设置测试

在语音交互界面中：
- 调整 **音量** 滑块
- 调整 **语速** 滑块
- 调整 **音调** 滑块
- 选择不同的 **语音类型**

### 语音合成历史

1. 点击"📜 语音合成历史"标签
2. 查看历史合成记录
3. 点击播放按钮重新播放历史音频

### 录音历史

1. 点击"🎙️ 录音历史"标签
2. 查看历史录音记录
3. 重新播放或重新识别录音内容

## 🐛 故障排除

### 常见问题

1. **麦克风权限问题**
   - 确保浏览器有麦克风权限
   - 检查系统麦克风设置

2. **音频播放失败**
   - 检查系统音量
   - 确认浏览器音频权限

3. **Live2D 角色不显示**
   - 检查浏览器控制台错误信息
   - 右键角色启用调试模式

4. **语音识别失败**
   - 确认网络连接
   - 检查腾讯云配置是否正确

### 调试方法

1. **启用调试模式**：右键角色 → 开启调试模式
2. **查看控制台**：打开浏览器开发者工具查看详细日志
3. **检查网络请求**：在 Network 标签中查看 API 调用
